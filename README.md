# 🪞 MIRROR — Behavioral Finance Coach

## 🔴 Problem Statement
Retail investors in India lose billions annually due to emotional biases. Platforms are optimized for trade volume, not investor safety. MIRROR intervenes during "Panic Selling" and "FOMO Buying" to protect capital.

## 💡 Solution Overview
An AI-powered coach that detects behavioral anomalies using Isolation Forest scores. It triggers a **BiasBrake** intervention, forcing a 24-hour reflection period and showing backtested "What-If" outcomes.

## 🛠 Tech Stack
- **Frontend:** Vanilla HTML5, CSS3 (Custom Properties), JavaScript (ES6+).
- **Visualization:** HTML5 Canvas API (Radar & Trend charts).
- **ML Logic:** Client-side behavioral rule-engine (simulating Isolation Forest).
